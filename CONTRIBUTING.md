# 贡献指南
如果你想为 Marketing for Founders 做出贡献，请提交一个 Pull Request。请遵循以下准则：
* 在提出新建议之前，请先搜索之前的建议，因为你的建议可能是重复的。简要检查所选类别中的内容，以确保你的建议不会与现有的重复。
* **为每条建议创建一个单独的 Pull Request。**
* 尽量让你的 Pull Request 内容详尽。说明这个资源或工具对你有什么帮助。我们希望你推荐的内容是你亲自读过、看过、听过或使用过的。
* 将你提交的内容放在对应分类的末尾。
* 欢迎提出新的分类或对现有分类的改进！
* 所有描述都以句号结尾，并删除末尾的空格。
* 请检查你的拼写和语法。

## 链接格式
在 Pull Request 中，请使用以下链接格式：
* `[文章标题](链接)`。
如果需要，可以随时更改标题以提高清晰度/使其更合适。
* `[工具名称](链接)`。说明该工具对你的帮助，以及它是否提供免费试用或免费功能。


## 内容要求
* 你提交的内容或工具应该对其他创始人有用。显然，它必须与市场营销、销售、公关或增长黑客任务相关。
* 内容应该是最新的、文笔优美、结构清晰。
* 视频和播客应为英文，以便全球任何人都能理解！
* 请不要提交无耻的推广内容。
* 内容以软件为重点是加分项，但不是必需的。
* 工具可以是付费的，但应有针对个人的定价计划。请不要提交面向营销机构且费用高昂的工具。

## 审核流程
我会尽我所能尽快审核你提交的内容。但是，我希望能深入了解它，这可能需要一些时间。

如果内容或工具未通过筛选，你提交的贡献可能会被拒绝。这可能是由于多种原因：文笔差、过多的自我推销、与主题无关等。
